Tucarro.com.co

Un componente web es una pieza reutilizable de interfaz construida con HTML y CSS. A continuación ves un ejemplo real: una galería de imágenes de vehículos, similar a la que usaría un portal como Tucarro. Cada tarjeta de imagen es un componente independiente con su contenedor .img, su imagen <img> y efectos de hover aplicados por CSS. Así se construyen las interfaces que ves en internet todos los días.

¿Cómo se construye este componente?

Cada imagen está envuelta en un <div class="img"> que actúa como contenedor con overflow: hidden para recortar la imagen. La imagen tiene object-fit: cover para llenar el espacio sin deformarse. El efecto de escala al pasar el cursor usa transform: scale(1.1) con transition. El conjunto usa Flexbox con flex-wrap: wrap para distribuirse en filas según el ancho disponible.
