Propiedades CSS

¿Qué es una propiedad?

Una propiedad CSS es una instrucción que le dice al navegador cómo debe verse un elemento. Cada propiedad tiene un nombre y un valor, separados por dos puntos: propiedad: valor;. Son el vocabulario del lenguaje CSS: sin ellas, toda página web sería texto negro sobre fondo blanco.

Propiedades esenciales por categoría

Tipografía: font-size controla el tamaño del texto. font-family define la fuente (Arial, Helvetica, etc.). font-weight regula el grosor (bold, 100–900). line-height ajusta el espacio entre líneas para mejorar la legibilidad. letter-spacing controla el espacio entre caracteres.

Color y fondo: color cambia el color del texto. background-color define el fondo del elemento. Los valores pueden ser nombres (red), hexadecimales (#FF0000), o funciones como rgb(255, 0, 0) y rgba() que también admiten transparencia.

Espaciado y borde: margin controla el espacio exterior de la caja. padding controla el espacio interior. border define grosor, estilo y color del contorno. border-radius redondea las esquinas del elemento.

Efectos visuales: box-shadow añade sombra a los contenedores. text-shadow aplica sombra al texto. opacity controla la transparencia del elemento completo. transition hace que los cambios de estilo ocurran de forma animada y suave.
