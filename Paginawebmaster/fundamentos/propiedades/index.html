Propiedades

Aprende que es una propiedad

Una propiedad de CSS es un atributo que se le da a un estilo en especifico, tales como colores, tamaños de letra, margenes, entre otros. Las propiedades controlan la aparencia visual del sitio web y le dan personalidad.
