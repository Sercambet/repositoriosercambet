Flexbox en Acción

Los cuadrados negros de abajo son flex items dentro de un contenedor con display: flex y flex-wrap: wrap. Observa cómo se distribuyen automáticamente en filas según el ancho disponible. Prueba a cambiar el tamaño de la ventana para ver el comportamiento responsivo.

Cada caja mide 20x20px. El contenedor usa gap: 20px para el espaciado y justify-content: center para centrar el grupo de cajas horizontalmente.
