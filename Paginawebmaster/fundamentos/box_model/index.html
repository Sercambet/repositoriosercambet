Box Model

En CSS, Boxmodel describe como todos los elementos HTML se renderizan en un cuadrado, como este.

A través de HTML, podemos ingresar información que se ajusten en las cajas, tales como texto, imagenes entre otros.

Adicionalmente, es posible brindarles caracteristicas de margen como border y padding para que la información se vea correctamente acomodada.
