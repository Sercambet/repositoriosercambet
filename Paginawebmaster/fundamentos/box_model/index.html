Box Model

En CSS, el Box Model describe cómo cada elemento HTML ocupa espacio en la pantalla. Todo elemento se representa como una caja rectangular compuesta por cuatro capas: content, padding, border y margin.

Content (Contenido): Es el área donde vive el texto, la imagen o cualquier elemento interno. Su tamaño se controla con las propiedades width y height.

Padding (Relleno interior): Es el espacio entre el contenido y el borde de la caja. Aumentarlo hace que el contenido "respire" dentro del elemento sin afectar elementos externos.

Border (Borde): Es la línea que rodea el padding y el contenido. Puedes definir su grosor, estilo (solid, dashed, dotted) y color con la propiedad border.

Margin (Margen exterior): Es el espacio entre la caja y los elementos que la rodean. El margen es transparente y empuja otros elementos lejos de la caja actual.

La propiedad box-sizing: border-box es clave en el desarrollo moderno: hace que el padding y el border se calculen dentro del width definido, evitando que la caja crezca de forma inesperada.
