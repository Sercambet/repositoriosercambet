Selectores CSS

Un selector CSS es la parte de una regla que indica a qué elemento HTML se le aplicará el estilo. Sin selectores, no hay forma de decirle al navegador qué parte de la página debe cambiar de color, tamaño o posición.

Los tres tipos fundamentales son:

  • Selector de etiqueta: apunta a todas las etiquetas de ese tipo. Ej: p { color: red; } afecta todos los párrafos.
  • Selector de clase (.nombre): apunta a elementos con ese atributo class. Es el más usado en proyectos reales.
  • Selector de ID (#nombre): apunta a un único elemento con ese id. Se usa para elementos únicos en la página.

En los ejemplos a continuación, cada tarjeta de receta tiene una clase diferente (.card_brasil, .card_argentino, .card_sarten), lo que permite aplicarle un color de fondo distinto a cada una sin tocar las demás. Así funcionan los selectores de clase en la práctica.

Como asar carne en sarten

  • 300 gr. de falda o carne para sopa (se conoce también como vacío, matambre o tapabarriga) (aprox. 150 gr. por persona)
  • ½ cebolla mediana
  • ¼ de pimentón (pimiento) rojo mediano
  • 1 zanahoria mediana
  • 2 cucharadas de aceite de oliva
  • ½ cucharada de comino molido
  • ½ cucharada de pimentón dulce molido
  • ½ cucharada de pimentón ahumado molido
  • ½ taza de agua (también puedes utilizar caldo de pollo o caldo de pollo o vegetales)
