Selectores CSS

Un selector CSS es la parte de una regla que indica a qué elemento HTML se le aplicará el estilo. Sin selectores, no hay forma de decirle al navegador qué parte de la página debe cambiar de color, tamaño o posición.

  • Etiqueta: apunta a todas las etiquetas de ese tipo. Ej: p { color: red; }
  • Clase (.nombre): apunta a elementos con ese atributo class. Es el más usado en proyectos reales.
  • ID (#nombre): apunta a un único elemento con ese id. Para elementos únicos en la página.

Cada tarjeta tiene una clase diferente (.card_brasil, .card_argentino, .card_sarten), lo que permite aplicarle un color de fondo distinto a cada una. Así funcionan los selectores de clase en la práctica.
