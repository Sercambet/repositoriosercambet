Flexbox

Esta layout de CSS permite que nuestro diseño sea flexible, compacto y práctico para que se mantenga en linea con todas las columnas, contenedores, cajas, etc. Sus mayores y mejores atributos son "justify-content", "flex-wrap" "content-align" entre otros. Flexbox permite simplificar multiples y complejos diseños para asegurar que la página web sea navegable y coherente visualmente.
