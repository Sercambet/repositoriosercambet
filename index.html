La Biblioteca

Bienvenido a Codigocraft.com, la guía de HTML y CSS diseñada para llevarte de cero a desarrollador web con confianza. Aquí encontrarás cada concepto explicado con claridad, ejemplos visuales y un lenguaje directo, sin rodeos. Cada módulo cubre un pilar fundamental del desarrollo front-end: desde cómo se estructura una página hasta cómo darle estilo, animación y responsive design. Tómate tu tiempo, explora en orden o salta al tema que necesites. Esta biblioteca está hecha para ti.

¿No sabes por dónde empezar? Te recomendamos este orden: Box Model → Selectores → Propiedades → Medidas → Flexbox → Componentes. Cada sección construye sobre la anterior y al final tendrás las bases sólidas para construir cualquier interfaz web.

  • 📦 Box Model

Todo elemento HTML vive dentro de una caja invisible. Aprende a controlar su contenido, padding, borde y margen para construir layouts precisos y ordenados.

  • 🧩 Componentes

Los componentes son bloques reutilizables de HTML y CSS. Desde una tarjeta hasta un menú de navegación completo: aprende a encapsular y estructurar interfaces modulares.

  • ↔ Flexbox

Flexible Box Layout es el sistema de distribución de CSS más poderoso para organizar elementos en fila o columna. Domina justify-content, align-items y flex-wrap en esta sección.

  • 📐 Medidas

Los px son solo el comienzo. Descubre cómo funcionan rem, em, %, vh y vw, y cuándo usar cada unidad para que tu diseño sea realmente adaptable a cualquier pantalla.

  • 🎨 Propiedades

CSS tiene cientos de propiedades que controlan colores, tipografía, sombras, fondos, bordes y más. Aprende las más importantes y cómo combinarlas para crear páginas con identidad visual sólida.

  • 🎯 Selectores

Los selectores son la llave de CSS: te permiten apuntar exactamente al elemento que quieres estilizar. Aprende selectores de etiqueta, clase, ID y combinadores para tener control total sobre tu diseño.
