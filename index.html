La Biblioteca

Bienvenido a la guía de código HTML y CSS más grande y completa de internet. Aquí encontrarás conocimiento de el mundo del desarollo web, donde abarcaremos toda una guía completa de fundamentos teóricos que funcionarán para crear tu página web perfecta. Ten en cuenta que cada página está escrita por expertos con cientos de horas de conocimiento, personas que trabajan para Google, Oracle, Adobe, entre otros. Te invitamos a ir por un café, relajarte y disfrutar de esta biblioteca de conocimiento inspirada en la construcción modular al estilo Minecraft, pero de desarrolladores.

  • Box Model

En esta sección aprenderás a modelar cajas de .CSS para lograr diseños como el que estás viendo, para dividir, agrupar y organizar conjuntos de información tales como texto, imagenes, enlaces entre otros.

  • Componentes

Aprende a controlar componentes, encapsular elementos, y utilizar tecnologías de lenguaje de codificación tales como HTML y CSS para la creación de menús.

  • Flexbox

Flexible Box Layout es propiedad del modelado de CSS que permite ajustar los items en una sola dirección, bien sea en row o column según corresponda. Aprende como utilizar esta gran herramienta en esta sección.

  • Medidas

Te vas a dar cuenta que la medida "px" es solo la punta del iceberg cuando nos referimos a medir tamaños. Encuentra y conoce las diferentes dimesiones que se utilizan al codificar CSS.

  • Propiedades

CSS posee multiples funciones para darle personalidad a tu página web, dentro de ellas están los colores, el tamaño de letra, margenes, fondos de color, entre otros. Aprende toda la nomenclatura de este lenguaje que le da vida a tu sitio en internet.

  • Selectores

¿Te has preguntado como es posible personalizar uno a uno cada texto, imagen, fondo y contenido que ves al detalle? Bueno, los selectores permiten identificar cada pixel que vas a modificar en tu código CSS y HTML, descubre como hacerlo.
